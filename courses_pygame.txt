## 1. Installation
Installer Pygame avec la commande :
		pip install pygame

## 2. Initialisation de Pygame
```python
import pygame
pygame.init()
```

## 3. Création de la fenêtre
```python
window = pygame.display.set_mode((largeur, hauteur))
pygame.display.set_caption("Titre de la fenêtre")
```

## 4. Boucle principale (game loop)
```python
running = True
while running:
		for event in pygame.event.get():
				if event.type == pygame.QUIT:
						running = False
		# ... (dessins et logique)
		pygame.display.flip()  # ou pygame.display.update()
```

## 5. Affichage et dessin
- Remplir l'écran :
	```python
	window.fill((0, 0, 0))  # noir
	```
- Dessiner un rectangle :
	```python
	pygame.draw.rect(window, (255, 0, 0), (x, y, largeur, hauteur))
	```
- Dessiner un cercle :
	```python
	pygame.draw.circle(window, (0, 255, 0), (x, y), rayon)
	```
- Afficher une image :
	```python
	image = pygame.image.load("chemin/vers/image.png")
	window.blit(image, (x, y))
	```

## 6. Gestion du temps
```python
clock = pygame.time.Clock()
clock.tick(60)  # Limite à 60 FPS
```

## 7. Gestion des événements
- Quitter : voir boucle principale
- Clavier :
	```python
	keys = pygame.key.get_pressed()
	if keys[pygame.K_LEFT]:
			# action à gauche
	```
- Souris :
	```python
	pos = pygame.mouse.get_pos()
	bouton = pygame.mouse.get_pressed()
	```

## 10. Sprites et groupes de sprites
Pour gérer plusieurs objets (drones, obstacles, etc.) :
```python
class Drone(pygame.sprite.Sprite):
		def __init__(self, x, y):
				super().__init__()
				self.image = pygame.Surface((20, 20))
				self.image.fill((0, 0, 255))
				self.rect = self.image.get_rect()
				self.rect.topleft = (x, y)
		def update(self):
				# logique de déplacement
				pass

all_drones = pygame.sprite.Group()
drone1 = Drone(100, 100)
all_drones.add(drone1)

# Dans la boucle principale :
all_drones.update()
all_drones.draw(window)
```

## 13. Structure type d’un programme Pygame
```python
import pygame
pygame.init()
window = pygame.display.set_mode((800, 600))
clock = pygame.time.Clock()
running = True
while running:
		for event in pygame.event.get():
				if event.type == pygame.QUIT:
						running = False
		window.fill((0, 0, 0))
		# Dessins et logique ici
		pygame.display.flip()
		clock.tick(60)
pygame.quit()
```

## 14. Méthodes et classes principales à connaître
- pygame.Surface : images, dessins
- pygame.Rect : positions, collisions
- pygame.sprite.Sprite, pygame.sprite.Group : objets dynamiques
- pygame.event : gestion des événements
- pygame.time.Clock : gestion du temps
- pygame.draw : primitives graphiques
- pygame.key, pygame.mouse : entrées utilisateur
- pygame.mixer : sons

## 15. Méthodes à implémenter dans une classe héritant de pygame.sprite.Sprite

Pour associer tes classes à Pygame et profiter de la gestion automatique, il faut :

- Hériter de `pygame.sprite.Sprite`.
- Définir au minimum :
	- `self.image` : la surface à afficher (obligatoire pour draw)
	- `self.rect` : le rectangle de position/collision (obligatoire pour draw et collision)
- Implémenter la méthode `update(self, *args)` pour la logique de déplacement ou d’état (appelée automatiquement par les groupes)

### Exemple de structure minimale :
```python
class MonObjet(pygame.sprite.Sprite):
	def __init__(self, x, y):
		super().__init__()
		self.image = pygame.Surface((largeur, hauteur))
		self.image.fill((r, v, b))  # couleur
		self.rect = self.image.get_rect(topleft=(x, y))
		# autres attributs spécifiques

	def update(self, *args):
		# Logique de déplacement ou d’état
		pass

	def custom_method(self):
		# Méthode personnalisée pour ton objet
		pass
```

### Méthodes utiles à surcharger ou utiliser :
- `update(self, *args)` : appelée à chaque frame par le groupe (logique, déplacement, animation)
- `kill(self)` : retire le sprite du groupe
- `add(self, *groups)` : ajoute le sprite à un ou plusieurs groupes
- `remove(self, *groups)` : retire le sprite de groupes
- `spritecollide(sprite, group, dokill)` : collision entre un sprite et un groupe
- `groupcollide(group1, group2, dokill1, dokill2)` : collision entre groupes

### Exemple d’intégration dans la boucle principale :
```python
all_sprites = pygame.sprite.Group()
objet = MonObjet(50, 50)
all_sprites.add(objet)

while running:
	# ... gestion des événements ...
	all_sprites.update()
	window.fill((0,0,0))
	all_sprites.draw(window)
	pygame.display.flip()
```

---
En suivant ce modèle, tu peux facilement intégrer tes classes existantes à Pygame et déléguer l’affichage, la gestion des groupes et des collisions à la bibliothèque.
